Appearance

Agents

Agents shown when launching a new instance. At least one must stay enabled.

SetupScreen.tsx has no separate mobile layout or iOS-style grouped list: it's the exact same card component as the desktop Locations/Agents screen, just capped at max-w-520px instead of 820px. The only real differences on mobile are additive, not restyled: an extra "Appearance" section (Light/Dark/System, shown only via the showThemePicker prop) and the LAN-access/Remote-access sections being omitted entirely (they only render when !isMobile). Agents are plain checkboxes, not toggle switches, matching the desktop form exactly. There is no "Check for updates" row in this screen; update-checking lives in its own UpdateScreen.
