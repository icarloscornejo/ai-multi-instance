Reconnecting to the local server
The connection to the app's local server was lost, likely because it briefly restarted during a self-update. Your instances are unaffected, and this will resolve on its own.
Retrying in 3 seconds...
Replaces a raw fetch error with plain-language framing, plus a live countdown ring that ticks down each retry cycle while the local server restarts mid self-update. There is no manual retry button and no separate "still can't reach it" state: the screen just keeps auto-retrying with a fresh countdown each cycle until the connection returns.
